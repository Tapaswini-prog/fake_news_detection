# 📰 Fake News Detection App

## 🚀 Overview
This project detects whether a news article is **REAL or FAKE** using Natural Language Processing (NLP) and Machine Learning.

---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes
- Streamlit (for UI)

---

## ⚙️ Workflow
1. Load dataset (Fake.csv, True.csv)
2. Preprocess text data
3. Convert text using TF-IDF
4. Train model (Naive Bayes)
5. Evaluate model
6. Build Streamlit app
